#Lista de comandos
====================

### Comandos de VIM

1. $ vim nombredelarchivo.ext
2. Guardar Archovo : ESC + : + w
3. Guardar y salir : ESC + : + wq

### Comandos de Git

1. Iniciar un espacio de trabajo con Git
   $ git init
2. Checar el status
   $ git status
3. Crear una nueva rama
   $ git branch nombrederama
4. Cambiar de Rama
   $ git checkout nombrederama
